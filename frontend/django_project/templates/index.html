Backend

FastAPI is available at http://localhost:8000.

Use these example routes:

  • POST /api/v1/auth/register
  • POST /api/v1/auth/token
  • GET /api/v1/users/me
  • POST /api/v1/users/{user_id}/snapshots
  • POST /api/v1/users/{user_id}/reports/generate
